
# ❄️ Frota Climatização - Landing Page

Esta é a página oficial da **Frota Climatização**, focada na conversão de clientes para serviços de instalação, manutenção e higienização de ar-condicionado nas regiões de Belo Horizonte, Contagem e Betim.

A página foi desenvolvida com um design moderno, responsivo e otimizado para proporcionar uma experiência fluida tanto em desktop quanto em dispositivos móveis.

## 🚀 Tecnologias Utilizadas

O projeto utiliza tecnologias web fundamentais para garantir leveza e velocidade:

* **HTML5**: Estrutura semântica do conteúdo.
* **CSS3**: Estilização personalizada com uso de variáveis (`:root`), Flexbox e CSS Grid para layout responsivo.
* **JavaScript (Vanilla)**: Script leve para manipulação dinâmica do ano no rodapé.

## 📂 Estrutura de Arquivos

* `index.html`: Contém toda a estrutura da página, incluindo a seção Hero, depoimentos de clientes e integração com WhatsApp.
* `style.css`: Define a identidade visual com paleta de cores em tons de azul, branco e verde (padrão WhatsApp).
* `js.js`: Pequeno script que atualiza automaticamente o ano do copyright no rodapé.
* `/images`: Diretório destinado às imagens dos serviços e à logo da empresa.

## ✨ Funcionalidades Principais

1. **Chamada para Ação (CTA)**: Botões estrategicamente posicionados para direcionar o usuário diretamente ao WhatsApp da empresa.
2. **Seção de Serviços**: Apresentação visual dos serviços de Instalação, Manutenção e Higienização com suporte a vídeos demonstrativos.
3. **Prova Social**: Seção de depoimentos para gerar confiança em novos clientes.
4. **Design Responsivo**: Layout que se adapta automaticamente a diferentes tamanhos de tela (celulares, tablets e PCs).
5. **Botão Flutuante**: Um acesso rápido ao WhatsApp que acompanha a rolagem da página.

> **Nota**: Este projeto foi configurado para trabalhar com as principais marcas do mercado, como LG, Samsung, Midea, Springer, Gree e Electrolux.
